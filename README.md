# Library Management System

Приложение для управления библиотекой, реализующее работу с базой данных: хранение, поиск и обработку информации о книгах и пользователях.

---

## Описание

Проект представляет собой систему управления библиотекой, позволяющую:

- хранить данные о книгах
- учитывать пользователей
- управлять процессом выдачи и возврата книг

Разработан в рамках курсовой работы по дисциплине баз данных.

---

## Основные возможности

- Добавление и удаление книг
- Поиск книг по различным параметрам
- Управление пользователями
- Выдача книг
- Возврат книг
- Работа с базой данных (CRUD операции)

---

## Технологии

| Компонент | Технологии |
|-----------|------------|
| Язык | C++ |
| База данных | SQL (SQLite / MySQL / PostgreSQL) |
| Работа с БД | SQL-запросы |
| Архитектура | Клиент → База данных |

---

## Что реализовано

- Проектирование структуры базы данных
- Создание таблиц и связей
- Реализация CRUD-операций
- Написание SQL-запросов
- Обработка пользовательских действий
- Логика учёта книг и пользователей

---

## Архитектура системы

Система состоит из:

- **Слой данных** — база данных (таблицы книг, пользователей, операций)
- **Бизнес-логика** — обработка операций (выдача, возврат, поиск)
- **Интерфейс** — взаимодействие с пользователем
